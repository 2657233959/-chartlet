## 介绍
```
一个贴图控件, 仿InShot视频剪辑的贴图实现, 支持以下功能,如需二次开发,直接下载源代码修改即可.
```

## 效果图
![image](https://github.com/jack-chong/chartlet/blob/master/app/gif/chartlet.gif)





## 功能
+ 拉伸
+ 拖拽排序
+ 贴图有图片, 文字, 图片加文字三种类型
+ 可左右上下滑动
+ 新增自动排序
+ 松手自动定位到中间
+ 吸附效果
+ 手指移动到边界自动滚动
+ 超过顶部松手新增一行
+ 点击,长按
+ 底部导航条显示位置
